## 🚀 使い方

このリポジトリをクローンして、サンプルコードを実行することで、Gaiahライブラリの使い方を学ぶことができます。以下の手順に従ってください:

1. リポジトリをクローンする:
   ```
   git clone https://github.com/Sunwood-ai-labs/Gaiah_Sample02.git
   ```

2. プロジェクトディレクトリに移動する:
   ```
   cd Gaiah_Sample02
   ```

3. 必要な依存関係をインストールする:
   ```
   pip install gitpython
   ```

4. サンプルコードを実行する:
   ```
   python sample_code.py
   ```
